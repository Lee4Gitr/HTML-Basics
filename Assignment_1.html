Another quick lesson, most IDEs or text editors will have a special shortcut... if you type "lorem"... then press tab... This should allow you to generate a bunch of text in latin. It's standard practice to see test websites that use this to show things like font size, color, line spacing, etc. but not have any significant meaning. (It actually doesn't mean anything, even in Latin... and it's pretty old!)

Finally... for placeholder images, you can use this url (https://place-puppy.com/300x300) in the img tags "src" attribute. The sizing of the image is in pixels and at the end of the url, so it looks like https://place-puppy.com/WidthxHeight. Go to here and read if you have any questions.
